# javaRPSGame
スッキリわかるjava入門で学習した内容を利用して作成したじゃんけんゲーム
JDK17で開発

mapでグーチョキパーを数字に対応させて、Scannerでグーチョキパーを入力して、randomで相手の手を確定して勝ち負けを判断する。
Scannerで受け取った文字列がグー、チョキ、パー以外だとループするようになっています。

## 使い方
JDKをインストールの上
ターミナルやコマンドプロンプトでコンパイル
ターミナルやコマンドプロンプト上で実行

## 実行するまでの手順
ターミナルやコマンドプロンプトのカレントディレクトリにファイルを配置
ターミナルやコマンドプロンプト上で `javac RPSGame.java` でコンパイル後
ターミナルやコマンドプロンプト上で `java RPSGame` で実行
